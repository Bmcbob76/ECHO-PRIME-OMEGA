QUICK START:
============

1. Install dependencies (ONE TIME):
   E:\ECHO_XV4\MLS\agents\reinstall_deps.bat

   E:\ECHO_XV4\MLS\agents\run_phd_agent.bat


TROUBLESHOOTING:
================

ERROR: Version conflicts / Import errors
SOLUTION: Run reinstall_deps.bat to fix dependency versions

ERROR: OPENAI_API_KEY not found
SOLUTION: Check E:\ECHO_XV4\CONFIG\echo_x_complete_api_keychain.env exists

ERROR: Module not found
SOLUTION: Reinstall dependencies with reinstall_deps.bat


MANUAL INSTALLATION:
====================

If reinstall_deps.bat fails, run manually:

H:\Tools\python.exe -m pip uninstall -y openai pydantic typing-extensions
H:\Tools\python.exe -m pip install --upgrade pip
H:\Tools\python.exe -m pip install openai>=1.12.0
H:\Tools\python.exe -m pip install pydantic>=2.6.0,<3.0.0
H:\Tools\python.exe -m pip install psutil>=5.9.0
H:\Tools\python.exe -m pip install pywin32>=306
H:\Tools\python.exe -m pip install typing-extensions>=4.9.0
